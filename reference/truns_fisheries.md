# Returns a dataframe with fisheries defined by the TRuns report driver

Returns a dataframe with fisheries defined by the TRuns report driver

## Usage

``` r
truns_fisheries(fram_db)
```

## Arguments

- fram_db:

  FRAM database object

## See also

[`truns_stocks()`](https://framverse.github.io/framrsquared/reference/truns_stocks.md)

## Examples

``` r
if (FALSE) truns <- truns_fisheries(fram_db) # \dontrun{}
```
