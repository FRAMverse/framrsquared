# Compares the stock fishery rate scalers of two runs

Compares the stock fishery rate scalers of two runs

## Usage

``` r
compare_stock_fishery_rate_scalers(fram_db, run_ids)
```

## Arguments

- fram_db:

  FRAM database object

- run_ids:

  Two run ids

## See also

[`compare_runs()`](https://framverse.github.io/framrsquared/reference/compare_runs.md)

## Examples

``` r
if (FALSE) fram_db |> compare_stock_fishery_rate_scalers(c(55, 56)) # \dontrun{}
```
