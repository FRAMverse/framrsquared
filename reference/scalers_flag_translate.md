# Provides English translation of numeric scalers flags

Provides English translation of numeric scalers flags

## Usage

``` r
scalers_flag_translate(vec)
```

## Arguments

- vec:

  vector of flags

## Examples

``` r
if (FALSE) scalers_flag_translate(sample(c(1, 2, 7, 8, 17, 18, 27, 28), 10, replace = T)) # \dontrun{}
```
