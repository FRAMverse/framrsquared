# NA's all the information in the FisheryScalers that's not being used e.g Flag 1 only NS Scalers will be returned

NA's all the information in the FisheryScalers that's not being used e.g
Flag 1 only NS Scalers will be returned

## Usage

``` r
filter_flag(.data)
```

## Arguments

- .data:

  Fishery Scalers table

## Examples

``` r
if (FALSE)  fishery_scalers_table |> filter_flag() # \dontrun{}
```
