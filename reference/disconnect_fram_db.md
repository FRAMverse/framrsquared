# Safely disconnect from FRAM database

Safely disconnect from FRAM database

## Usage

``` r
disconnect_fram_db(fram_db, quiet = TRUE)
```

## Arguments

- fram_db:

  FRAM database R object

- quiet:

  Logical. Optional; when true, silences success message.

## See also

[`connect_fram_db()`](https://framverse.github.io/framrsquared/reference/connect_fram_db.md)

## Examples

``` r
if (FALSE) disconnect_fram_db(fram_db) # \dontrun{}
```
