# Vectorized approach to calculating the management week, returns an integer

Vectorized approach to calculating the management week, returns an
integer

## Usage

``` r
management_week(date)
```

## Arguments

- date:

  A column with dates

## Examples

``` r
if (FALSE) { # \dontrun{
data_fram |>
  mutate(mngmt_week = management_week(date_field))
} # }
```
